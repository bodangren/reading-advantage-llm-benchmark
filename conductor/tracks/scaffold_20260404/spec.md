# Track Specification: Scaffold Next.js application and implement core layout

## Overview
Initialize the Next.js (App Router) project with TypeScript, Tailwind CSS, and shadcn/ui. Set up the foundational static JSON data structures and implement the core application shell (header, footer, layout).

## Requirements
- Use `create-next-app` with App Router and TypeScript.
- Integrate Tailwind CSS and initialize shadcn/ui.
- Create mock static JSON files for early development.
- Implement a responsive layout with dark mode support.
