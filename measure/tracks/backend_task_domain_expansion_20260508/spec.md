# Spec: Backend/API Task Domain Expansion

## Goal
Expand the benchmark catalog beyond frontend/web tasks to include backend and API integration tasks: database schema design, API route implementation, authentication middleware, and service refactoring.

## Acceptance Criteria
- [ ] At least 5 new backend tasks added to the task catalog
- [ ] Tasks cover: DB schema migration, REST API endpoint, tRPC/router, auth middleware, service extraction/refactor
- [ ] Each task has a real repo snapshot with existing backend code
- [ ] Scoring rubric adapted for backend (type safety, test coverage, error handling, performance)
- [ ] Leaderboard and run detail pages correctly display backend task results

## Out of Scope
- DevOps/infrastructure tasks
- Mobile native tasks
