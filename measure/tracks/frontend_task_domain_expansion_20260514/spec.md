# Spec: Frontend/UI Task Domain Expansion

## Context
The `backend_task_domain_expansion_20260508` track added backend-focused tasks (DB migrations, REST APIs, auth middleware). This track extends the benchmark catalog to cover frontend and UI engineering tasks.

## Goal
Add 8-12 frontend/UI tasks to the benchmark catalog covering React components, CSS/styling, accessibility, and responsive design integration into existing codebases.

## Acceptance Criteria
- [ ] Task definitions include repo snapshots with existing UI codebases
- [ ] Scoring rubric adapted for visual/UI correctness (pixel diff, a11y audit, interaction tests)
- [ ] Tasks added to `tasks.json` with proper metadata
- [ ] At least one task uses the existing reading-advantage or primary-advantage repo snapshot
- [ ] Harness verification steps validate UI output (build, visual regression, or component test)
