# Product Guidelines

## 1. Prose Style
**Principle:** Concise & Minimal
- Use bullet points and short sentences.
- Focus strictly on scannability and quick comprehension.
- Avoid marketing fluff or overly academic jargon.

## 2. User Experience (UX)
**Principle:** Simple & Actionable
- Prioritize quick insights and clear, high-level scores on the leaderboard and run pages.
- Ensure the primary user journey (finding a task, viewing top models) is frictionless.
- Avoid overwhelming the user with raw data unless explicitly requested (e.g., in a detailed drill-down).

## 3. Visual Branding
**Principle:** Modern DevTool
- Aesthetic should be sleek and developer-focused.
- Default to Dark Mode.
- Use a minimalist base with vibrant accent colors (similar to Vercel or Linear) to highlight key metrics and actions.
